COM2090SEF_Project
This project aims to let HKMU students and teachers to use for knowing wether the elevator passengner flow is high or low at the time they back school.
Therefore a linear regression is used in this project and letting user input time and have a prediction with model which has been trained.
For the furter improve, more model for example ole , jcc and mc model will be trained and with larger time range not only 10 min befor class start and 10 min after class start time.
